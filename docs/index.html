🚀 Quick Installation

Get started in less than 30 seconds

⚡ Quick Install (Recommended)

bash 
curl -sSL https://raw.githubusercontent.com/oiahoon/termonaut/main/install.sh | bash

🍺 Homebrew

bash

Install Termonaut
